# Digit Recognizer From Scratch (NumPy Only)

## Overview
This project implements a digit recognition model from scratch using only NumPy and Pandas, without relying on deep learning frameworks such as PyTorch or TensorFlow. The model is trained on the MNIST dataset and demonstrates how neural networks can be built and trained using fundamental numerical operations.

## Objectives
- Build a neural network from first principles
- Understand forward propagation, backpropagation, and gradient descent
- Gain low-level intuition of how deep learning models work

## Features
- Fully connected neural network implemented from scratch
- Manual implementation of:
  - Weight initialization
  - Activation functions (ReLU / Sigmoid / Softmax)
  - Loss function (Cross-Entropy)
  - Backpropagation
- Training loop with gradient descent
- Data handling using Pandas

## Tech Stack
- Python
- NumPy
- Pandas
- Matplotlib (optional for visualization)

## Dataset
- MNIST (handwritten digit dataset)
- 28x28 grayscale images (flattened to vectors)

## Results
- Achieves solid accuracy on MNIST without using any ML frameworks
- Demonstrates correctness of manual gradient computations
